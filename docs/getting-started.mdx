# Andromedocs

**Andromedocs** provides simple yet flexible tools for documenting Design Systems. This project and its documentation are currently in development and, therefore, are undergoing frequent changes.

## Features

- A ready-to-go design system documentation.
- Document using MDX and React.
- Built-in Components that helps you document your components.
- Use theme ui to customize your documentation.

## Getting Started

1. Add our theme to your gatsby project
   `yarn add andromedocs-theme`
2. Start documenting!

### Customize navbar

If you wish to customize the navbar, create a file `andromedocs-theme/src/data/sidebar.yaml` then you can customize your navbar following this structure:

```yaml
- label: "Introduction"
  link: "/"
- label: "Components"
  items:
    - label: "Guidelines"
      link: "/components/guidelines/"
    - label: "LiveCode"
      link: "/components/LiveCode/"
```

### Customize the theme

If you wish to customize the theme, create a file `andromedocs-theme/src/gatsby-plugin-theme-ui` and add your theme.
