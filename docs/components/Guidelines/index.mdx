# Guidelines

The Guidelines component describes how a component is expected to be used (Dos) or not (Don'ts).

## Example

<LiveCode
  scope={{ Guidelines }}
  code={`<Guidelines>
  <Guidelines.Dos>
    <li>Do this!</li>
    <li>Do that!</li>
  </Guidelines.Dos>
  <Guidelines.Donts>
    <li>NEVER do this!</li>
    <li>NEVER do that!</li>
  </Guidelines.Donts>
</Guidelines>`}
></LiveCode>
